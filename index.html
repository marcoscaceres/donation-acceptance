This specification allows a webpage to indicate the presence of a page for receiving donations. Anyone wishing to make a donation to the website can issue a request to this page, specifying the amount they wish to donate. The page will create a Web Payments Payment Request in the browser. User-agents are encouraged to create automated means of hitting this endpoint and accepting the payment request to make regular donations possible.

This is a good idea to getting artists and people paid, but this document unofficial.

Introduction

Web Payment Donate URI provides a way for a User-agent to realize & act upon a "donation" endpoint provided by a web page.

A web host which is setup for WebPayment Donation URI will provide a endpoint, which when given an 

amount
in the querystring, creates and shows Payment Request for that amount.

Well-Known URI

Page Link

Links in a page's headers may include a link with a link-relation of "webpayment-donate"
